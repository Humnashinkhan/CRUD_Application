# CRUD_Application Using React.Js
This repository contains a simple CRUD (Create, Read, Update, Delete) application built using React.js. 
This application demonstrates the basic functionality of managing a list of items, allowing users to create new items,
read the list of items, update existing items, and delete items.

Features

Create: Add new items to the list.
Read: View the list of items.
Update: Edit details of existing items.
Delete: Remove items from the list.

Usage
Add a new item:

Click on the "Add Item" button.
Fill in the details in the form and submit.
View items:

The list of items will be displayed on the main page.
Edit an item:

Click the "Edit" button next to the item you want to edit.
Modify the details in the form and submit.
Delete an item:

Click the "Delete" button next to the item you want to remove.
